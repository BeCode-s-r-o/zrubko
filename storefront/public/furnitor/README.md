# Furnitor Theme Files

## Directory Structure

Place your Furnitor theme files in the following directories:

- **css/** - All CSS files from Furnitor (bootstrap.min.css, furnitor-style.css, etc.)
- **js/** - All JavaScript files from Furnitor (bootstrap.bundle.min.js, furnitor-script.js, etc.)
- **images/** - All images, icons, and graphics from Furnitor
- **fonts/** - All font files from Furnitor

## How to Extract Files

1. Unzip your Furnitor theme package
2. Look for the `dist` or `assets` folder
3. Copy files as follows:
   - CSS files → `css/`
   - JavaScript files → `js/`
   - Images → `images/`
   - Fonts → `fonts/`

## Example Structure After Extraction

```
furnitor/
├── css/

│   ├── furnitor-style.css

│   ├── hero-bg.jpg
│   └── ...

    └── ...
```
